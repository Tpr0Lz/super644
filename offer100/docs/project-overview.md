# Offer100 项目概述与答辩要点

## 项目背景
当今社会科技与经济高速发展，人才需求变化快，就业信息呈现碎片化。Offer100 旨在建立企业与求职者的信息共享平台，帮助大学生与社会人士及时了解招聘趋势。

## 目标用户
- 在校大学生
- 社会求职者
- 企业招聘管理人员

## 已实现功能映射
1. 基于角色用户权限管理：
- JWT 登录认证
- 路由级权限控制（admin/student/social）

2. Socket 通信：
- 职位发布和投递行为实时广播
- 前端动态列表实时刷新

3. 基于用户行为数据统计分析：
- 记录用户浏览/发布/投递行为
- 管理端查看行为总量、按角色统计、按行为类型统计

4. 报表导出：
- 导出行为数据 CSV 报表

## 技术栈
- 前端：Vue3 + Vite + Pinia + Vue Router + Axios + Socket.IO Client
- 后端：Node.js + Express + JWT + Socket.IO
- 报表：CSV 导出（后续可替换 FreeReportBuilder）

## 可扩展方向
- 接入 MySQL 和 ORM
- 增加岗位搜索推荐算法
- 引入 AI 职业咨询接口（预留）
- 对接企业 HR 系统与消息通知渠道
